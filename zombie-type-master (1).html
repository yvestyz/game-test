about this game

這款遊戲誕生於千日文理教育 Vibe Coding Camp 的密集工作坊。製作者在課程開始前，從未碰過任何一行程式碼。


透過 Replit Agent 搭建遊戲架構、Claude 優化遊戲邏輯並平衡難度——從零到有，完整的殭屍打字闖關遊戲在工作坊內誕生。 遊戲加入全球即時排行榜，讓每一局都是真實的世界級競技。

game features
⌨️
打字消滅殭屍
每隻殭屍頭上懸掛一個英文單字，正確拼出即可擊殺。手指越快，殭屍死得越快。
🌊
波次系統
殭屍分批湧入，波次越高數量越多、速度越快，隨時保持緊張感。
⚡
字長決定速度
單字越長，殭屍移動越慢；短詞敵人移動迅猛，考驗反應速度。
🏆
全球排行榜
每局結束後分數即時上傳，與全球玩家同台競技，搶攻排名頂端。
🎨
像素風格
復古像素藝術美術風格，終端機綠色配色，打出經典 8-bit 末日氛圍。
🎯
三種難度
簡單、普通、困難三擋選擇，初學者輕鬆入門，老手挑戰極限。
how to play
01
選擇難度
進入遊戲後選擇 Easy / Normal / Hard，難度影響殭屍數量、移動速度與單字複雜度。
02
觀察殭屍頭上的單字
每隻殭屍頭頂懸掛一個英文單字，仔細看清楚目標，鍵盤開打。
03
快速正確地拼出單字
開始輸入第一個字母後，遊戲自動鎖定該殭屍。完整拼完即擊殺，輸入錯誤不扣分但要重新輸入。
04
守住生命值，挑戰高分
殭屍碰到邊界就會扣血，三條命用完遊戲結束。分數即時上傳全球榜，看看你能排第幾！
difficulty modes
Easy
簡單
單字短、殭屍少、速度慢，適合第一次挑戰或英文練習入門。
Normal
普通
均衡挑戰，單字長度與殭屍數量適中，體驗完整的打字節奏感。
Hard
困難
殭屍群湧、單字複雜、速度極快，手速與拼字雙重考驗，挑戰者限定。
